# Personal Expense, Budget & Financial Analytics System (MySQL)

This project tracks personal expenses and budgets using MySQL to
analyze spending patterns and financial behavior.
It categorizes expenses and compares actual spending with
defined monthly budgets.

The system helps users understand where money is spent and
supports better financial planning using data analytics.

## Technologies Used
- MySQL 8.0
- SQL

## SQL Concepts Covered
- Relational Tables
- Joins
- Aggregate Functions
- Date Functions
- Financial Analytics Queries

## Key Features
- Expense tracking by category
- Monthly budget comparison
- Spending analysis reports
- Financial insights using SQL

## Use Case
Useful for personal finance tracking, budgeting tools,
and financial analytics learning projects.
